##过滤器注意问题
1. vue2.0没有内置过滤器，而且自定义过滤器不能是实例化调用filter
是全局API 采用 Vue.filter来自定义

2 Vue.filter 要写在  new Vue()之前，否则会报 Failed to resolve filter：key的错
